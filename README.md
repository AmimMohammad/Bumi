# Cryptocurrency Arbitrage Trading Development


<h1>Trading Bot Development</h1>

<img src="https://imgur.com/3spc1QV.png" height="80%" width="80%" alt="diagram"/>

<h2>Objective<h2>
<b> Perform Trade Arbitrage in Price of the same ticker across multiple Exchanges. </b>

<h2>Description</h2>
In the Code below, We will Perform  : <br>
    <b> 1. Import Necessary Libraries and Modules</b><br>
    <b> 2. Enumerate key data from every exchange</b><br>
    <b> 3. Extract and Store common coin pairs</b><br>
    <b> 4. Check for Possible Arbitrage Opportunity</b><br>
    <b> 5. Calculate profit margin and print information about arbitrage opportunity</b><br>
